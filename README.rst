======================
acfun视频下载工具
======================


版本：1.1
作者：ylxy123
项目源代码地址：https://github.com/ylxy123/acfun-get


安装方法：
::
	pip install acfun-get


使用方法：(命令行)
::
	# 帮助
	>acfun-get -h

::
	# path为视频保存路径，默认为桌面，而且会记住上次设置的值
	# url为acfun视频链接，如https://www.acfun.cn/v/ac12345678
	>acfun-get -path xxx -url xxx
	# 提供所有可下载的清晰度
	# 路径参数-path/-p可以不指定，链接参数未指定时会提示输入
